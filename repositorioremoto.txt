Actividad IV: Repositorio Remoto
Ahora que hemos aprendido algunas habilidades geniales, vamos a usarlas creando y empujando nuestro propio repositorio GitHub.

 Crear un nuevo repo en GitHub llamado basic_repo
 Crear un repo local también llamado basic_repo
 Crear algunos archivos dentro de tu repo y agrega algo de contenido en ellos
 Agregar, confirmar y empujar los cambios a GitHub
 Hacer algunos cambios más en la copia local y empujarlos a GitHub
 Ve a tu cuenta de GitHub y mira tu nuevo repo y los cambios que has hecho
 Sube un archivo .txt simple con la descripción de los pasos que tomaste
